
# Task Board (Kanban App)

A customizable task management board built with React.

## 🔥 Features
- 📝 Add/Edit/Delete tasks
- 🎨 Status-based task coloring
- 🧲 Drag & Drop across columns
- ➕ Add/Delete columns dynamically
- 💾 LocalStorage persistence
